king's game

version:1.0.0 (Aug.26 2018)

キャンプやイベントで盛り上がっていると皆やりたがる（特に女子（偏見））、あの「王様ゲーム」をScratchで制作しました。
「追加」で新しい指令を追加できます。「開始」でEnterキーを押すと指令が10秒間表示されます。


prime fanctions

version:1.0.0 (Aug.29 2018)

素数関連の便利そうな関数を作ってみました。
nが素数かどうか判定するprime(n),
nまでの素数リストを生成するprimes(n),
nまでの双子素数リストを生成するdprimes(n)に分かれています。


SME(Simulator for Magicicadas Extermination)

version:3.0.0 (Aug.29 2018)

自然界では生存に有利なことが知られている「素数ゼミ」という蝉がいます。
ただ、有利すぎて1度に50億匹大量発生したりします。
そこで蝉が嫌いな僕が何とかPC上で絶滅させられないかと思い作ったシミュレーターです。
